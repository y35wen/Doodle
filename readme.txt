Description of the Game:

- This painting program allows users to custom drawing in their own ideas. The left side is a colour palette and a thickness choose that allows users to choose the properties of their drawing. The left down corner tracks current colour and width of the “pen”. 
   There is a menu bar on the top of the program, users can choose to start a new doodle, exit the program, or load or save a doodle.
